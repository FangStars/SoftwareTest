本程序实现了作业的基本功能和扩展功能，输入稍微有些许不同
使用方法：
wc.exe与file.c(必须)以及output.txt（必须）放在同一个文件夹内
本程序file.c为当不进行文件夹扫描时的默认输入文本
output.txt为默认输出位置
停用词文本可自定义命名，但输入时必须紧跟-e
输入文件和输出文件以及停用词表放在同一文件下可以直接使用
1.直接打开
  按提示输入即可
2.例如wc.exe file.c
  按提示进行下一步即可，只能输入一次指令，如-a
3.大于3个输入（3个输入判断功能是用来测试的，具体见代码）
  此时可输入wc.exe -c -w -o -s等类似指令
  注意：可不输入file.c 因为为默认的,输入也可，但只会扫描file.c
        -e 后紧跟停用词.txt
        -s 搜索的是和exe同目录全部c文件
        c文件要编码格式为UTF-8，否则会判断错误
